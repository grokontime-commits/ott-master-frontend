# Install Phase 3K — Customer Portal Frontend Integration

1. Keep backend running if you are testing:

```cmd
cd C:\OTT-Master-Backend
npm run dev
```

2. Unzip this package.
3. Copy all contents into:

```text
C:\Master-Frontend
```

4. Choose **Replace files in destination**.
5. Do not replace/delete:

```text
C:\Master-Frontend\live-server\ott-config.js
```

6. Hard refresh the browser with `Ctrl + F5`.
7. Open with Live Server:

```text
C:\Master-Frontend\live-server\ui\customer-portal.html
```

8. Login with your OFFICE/ADMIN test user for setup, or a CUSTOMER test user for read-only validation.

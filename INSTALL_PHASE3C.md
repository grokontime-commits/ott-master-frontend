# Install Phase 3C

1. Stop Live Server if it is running.
2. Unzip the Phase 3C package.
3. Copy the package contents into:

```text
C:\Master-Frontend
```

4. Choose **Replace files in destination**.
5. Keep your existing:

```text
C:\Master-Frontend\live-server\ott-config.js
```

6. Start backend:

```cmd
cd C:\OTT-Master-Backend
npm run dev
```

7. Open with Live Server:

```text
C:\Master-Frontend\live-server\ui\cargo-management.html
```

8. Test login, load catalogs, load MAWBs, select a MAWB, load detail, confirm HAWBs and status history.

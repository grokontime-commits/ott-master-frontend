# Install Phase 3I — Equipment Return Frontend Integration

1. Unzip this package.
2. Copy all contents into:

```text
C:\Master-Frontend
```

3. Choose **Replace files in destination**.
4. Do not replace or delete:

```text
C:\Master-Frontend\live-server\ott-config.js
```

5. Start backend:

```cmd
cd C:\OTT-Master-Backend
npm run dev
```

6. Open with Live Server:

```text
C:\Master-Frontend\live-server\ui\equipment-return.html
```

7. Login with your staging smoke test user and run the buttons in order.
